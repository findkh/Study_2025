# Sidebar 컴포넌트

Ant Design의 `Menu`를 활용하여 반응형 사이드바를 구현.  
기본적으로 메뉴는 접었다 펼 수 있으며, `collapsed` 상태를 조절할 수 있다.

## 📌 데이터 구조 (Menu Items)

```
const items = [
  { key: "1", icon: <PieChartOutlined />, label: "Option 1" },
  {
    key: "sub1",
    label: "Navigation One",
    icon: <MailOutlined />,
    children: [
      { key: "5", label: "Option 5" },
      { key: "6", label: "Option 6" },
    ],
  },
];
```
